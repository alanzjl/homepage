TARGET-TRACKING
MOBILE LAUNCHER

A mobile robot that tracks pre-defined target and automatically shoot ping-pong ball into it.

Overview

An automatic mobile launcher based on CV

In this project, we made a four-dimensional mobile robot which could track a pre-defined target and launch a ping-pong ball to hit it. Based on computer vision technology, the robot, mainly constructed by laser-printed Acrylic and 3-printed PLA material, could execute functions like target recognition, target tracking and launching trajectory planning.

MAIN FUNCTIONS

In this project, we made a mobile robot which could track a pre-defined target and launch a tennis ball to hit it. Once the robot is triggered into working condition, it will execute the following procedures:

Target recognition

Target-following

Justification of target’s static condition

Ball launching

Trajectory amendment

Due to the existence of the ball container, we could use a motor to control the popping frequency of the balls which might give the servo motor some time to adjust position of launching barrel when the target’s position changes.
